# AI-chatbot

The **JD University Chatbot** is a machine-learning based web chatbot built using **Python, Flask, HTML/CSS, and JavaScript**.
It answers university-related queries through predefined categories like *Admission, Courses, Fees, Campus, Scholarship,* and more.

The chatbot uses:

* **NLTK** for text preprocessing
* **CountVectorizer + Naive Bayes** for intent prediction
* **Flask API** for backend
* **SQLite** for saving chat history
